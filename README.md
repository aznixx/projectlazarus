# Lazarus Group Presentation

Een interactieve presentatie over de Noord-Koreaanse hackersgroep "Lazarus Group", gebouwd met SvelteKit en Tailwind CSS v4.

## Features
- **Cyber/Hacker Esthetiek**: Glitch effecten, matrix-achtige achtergronden, neon accenten.
- **Interactieve Slides**: Navigeer met de pijltoetsen of spatiebalk.
- **Animaties**: Vloeiende overgangen tussen slides.

## Installatie & Start

Dit project gebruikt `bun`.

```bash
bun install
bun run dev --open
```

## Navigatie
- **Volgende slide**: Pijl rechts (→) of Spatiebalk
- **Vorige slide**: Pijl links (←)

## Technologie
- SvelteKit (Svelte 5 Runes)
- Tailwind CSS v4
- Lucide Svelte Icons
